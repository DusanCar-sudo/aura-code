Capabilities

Why Aura Code

Aura doesn't try. Aura verifies. Every change is tested, every claim is checked. Built on the Praktess framework.

Multi-Model

Model-Agnostic

Switch between Claude, GPT, Gemini, MiMo, Ollama, or any OpenAI-compatible endpoint. One agent, any provider.

Autonomous

Autonomous Loop

Reads context, plans, executes tools, verifies results, and repeats until done. Full autonomous coding cycle.

Verification

Self-Verification

Every change triggers test runs. Regressions are caught instantly. 1035+ tests in the core test suite.

Providers

Multi-Provider

Resilient provider layer with fallback chains, circuit breakers, and rate limiting. Never blocked by API issues.

Telegram

Telegram Control

Control Aura remotely via Telegram. Send tasks, receive results, voice commands supported. Always connected.

Ruby

Ruby Fine-tuning

Built-in episode capture, training data generation, model selection, and fine-tuning pipeline. Self-improving agent.

How Aura Works

A modular, layered architecture from CLI input through orchestration, tool execution, to verified output.
